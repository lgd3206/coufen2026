【第1步】配置 Supabase 数据库 (10分钟)

  1. 打开浏览器访问: https://supabase.com
  2. 点击 "Start your project" 或 "Sign in"
  3. 使用 GitHub 账号登录
  4. 点击 "New Project"
  5. 填写项目信息:
     - Name: coufen-2026
     - Database Password: 设置强密码（记住！）
     - Region: Northeast Asia (Tokyo)
     - Pricing Plan: Free
  6. 点击 "Create new project"，等待2分钟

  7. 创建数据表:
     - 点击 "SQL Editor"
     - 点击 "New query"
     - 打开项目中的 SUPABASE_SETUP.md
     - 复制 SQL 代码并执行

  8. 配置 RLS 策略:
     - 按照 SUPABASE_SETUP.md 添加3个策略

  9. 启用 Realtime:
     - Database > Replication
     - 启用 submissions 表

  10. 获取 API Keys:
      - Project Settings > API
      - 复制 URL、anon key、service_role key

【第2步】部署到 Vercel (5分钟)

  1. 打开浏览器访问: https://vercel.com
  2. 使用 GitHub 账号登录
  3. 点击 "Add New..." > "Project"
  4. 选择 "Import Git Repository"
  5. 找到 "lgd3206/coufen2026"，点击 "Import"
  6. 配置环境变量（点击 "Environment Variables"）:

     NEXT_PUBLIC_SUPABASE_URL
     值: 你的 Supabase URL

     NEXT_PUBLIC_SUPABASE_ANON_KEY
     值: 你的 anon public key

     SUPABASE_SERVICE_ROLE_KEY
     值: 你的 service_role key

  7. 点击 "Deploy"
  8. 等待2-3分钟部署完成
  9. 获得 Vercel 域名: coufen2026.vercel.app
